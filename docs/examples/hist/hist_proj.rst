How to Project histograms
===========================

This example shows how to project a 2D histogram. 
To show the projections we use `matplotlib` to create a 3 by 1 grid to display multiple plots on the same window. The histograms are plotted using the plotting module `mpl` from this package (which internally uses matplotlib).
